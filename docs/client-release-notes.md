# Release Notes Client — MonprojetPro

## 2026-04-22 — Historique de l'étape en temps réel (Story 14.8)

**Ce qui change pour toi :** La colonne droite de chaque page d'étape affiche maintenant ton historique complet : tes soumissions avec leur statut coloré, tes documents générés et les feedbacks que MiKL t'a transmis. Le statut de tes soumissions se met à jour automatiquement sans recharger la page.

**Élio peut maintenant :**
- Afficher la liste de toutes tes soumissions pour une étape (avec badge En attente / Approuvé / Refusé / Révision)
- Montrer un aperçu de chaque soumission et son contenu complet en un clic
- Lister tes documents générés avec un bouton pour copier le contenu dans le presse-papier
- Afficher les feedbacks de MiKL avec un point bleu pour les messages non lus
- Mettre à jour le statut de tes soumissions en temps réel quand MiKL valide ou refuse

## 2026-04-22 — Nourrir Élio par étape (Story 14.6)

**Ce qui change pour toi :** Ton conseiller MiKL peut désormais injecter des précisions ou des documents directement dans le chat Élio d'une étape spécifique de ton parcours. La prochaine fois que tu ouvres le chat de cette étape, Élio t'annonce automatiquement les informations transmises.

**Élio peut maintenant :**
- Recevoir un prompt texte ciblé de MiKL (question, directive, précision)
- Recevoir un document (PDF, Word, TXT) joint par MiKL et l'annoncer au moment opportun
- Indiquer clairement que c'est MiKL qui a ajouté ces précisions ("MiKL a ajouté des précisions pour cette étape")
- Marquer automatiquement le contexte comme présenté après l'avoir communiqué
