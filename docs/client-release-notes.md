# Release Notes Client — MonprojetPro

## 2026-04-22 — Nourrir Élio par étape (Story 14.6)

**Ce qui change pour toi :** Ton conseiller MiKL peut désormais injecter des précisions ou des documents directement dans le chat Élio d'une étape spécifique de ton parcours. La prochaine fois que tu ouvres le chat de cette étape, Élio t'annonce automatiquement les informations transmises.

**Élio peut maintenant :**
- Recevoir un prompt texte ciblé de MiKL (question, directive, précision)
- Recevoir un document (PDF, Word, TXT) joint par MiKL et l'annoncer au moment opportun
- Indiquer clairement que c'est MiKL qui a ajouté ces précisions ("MiKL a ajouté des précisions pour cette étape")
- Marquer automatiquement le contexte comme présenté après l'avoir communiqué
